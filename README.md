# ElectroMart: Electronic Component Marketplace

ElectroMart is a Tier 2 Desktop GUI application designed for engineering students, academic staff, and university laboratory inventory managers to browse, search, purchase, and track electronic components efficiently.

## 🛠️ Technology Stack
* **Language:** Java SE
* **Frontend GUI:** Java Swing (utilizing CardLayout navigation)
* **Backend Database:** MySQL
* **Database Driver:** JDBC (Java Database Connectivity)
* **IDE:** IntelliJ IDEA

## 🧬 Core OOP Principles Implemented
This project is built to strictly adhere to robust Object-Oriented Design patterns:
* **Abstraction:** Abstract core models for unified system contracts (`User`, `Product`).
* **Inheritance:** Clean hierarchical structural reuse (e.g., `Semiconductor` extending `Product`).
* **Encapsulation:** Secure data state validation via strict `private` fields and structural `public` accessors.
* **Polymorphism:** Dynamic behavior overriding (e.g., component-specific `displayDetails()` implementation).
